# Content QA Checklist

Run this checklist before promoting any work to launch content.

- Source URL is recorded and reachable.
- Source library terms are compatible with Philo4All usage.
- Translator/editor is listed when applicable.
- Publication year and jurisdiction note are recorded.
- Public-domain or license status is documented.
- Chapter boundaries match the source.
- Reader formatting preserves paragraphs, headings, and section order.
- Work metadata includes author, tradition, era, and reading length.
- No copyrighted modern translation is included without permission.
- A human has read the imported text sample for obvious OCR or formatting errors.
