# Philo4All V1 Goal Workflow

## Active Goal Sequence

1. Foundation: monorepo, shared packages, Supabase setup, web/mobile shells.
2. Content: import pipeline, validation, and 3-5 verified seed works.
3. Reader: chapter navigation, typography controls, bookmarks, progress.
4. Account: Supabase auth, synced progress, synced bookmarks.
5. Library: browse, filters, work detail pages, search.
6. Design polish: cream/clay/oxblood system, responsive polish, accessibility.
7. Launch candidate: 10-20 works, QA, source audit, deployment/app readiness.

## Iteration Ritual

Each goal ends with:

- Working demo or artifact
- Test results
- Screenshot review for UI goals
- Known issues
- Decision: continue, revise, or promote

## Suggested Sub-Agent Roles

- Main agent: integration, product coherence, final technical decisions.
- Design agent: hierarchy, typography, responsive UI, screenshot critique.
- Engineering agent: architecture, schema, shared package boundaries.
- Content/legal agent: source provenance and public-domain audit.
- QA agent: acceptance tests, accessibility, launch blockers.
